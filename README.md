# Notes Management System

A full-stack Notes Management System built using React.js, Node.js, and Express.js.

## Features

- Create Notes
- View Notes
- Delete Notes
- REST API Integration
- Responsive UI
- Form Validation
- Separate CSS Styling

## Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js
- CORS

## Installation

### Clone Repository

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

---

## Backend Setup

Go to backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Run backend server:

```bash
npm run dev
```

Backend runs on:

```bash
http://localhost:5000
```

---

## Frontend Setup

Go to frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Run React app:

```bash
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```

---

## API Endpoints

### Get All Notes

```http
GET /notes
```

### Create Note

```http
POST /notes
```

### Delete Note

```http
DELETE /notes/:id
```

### Get Single Note

```http
GET /notes/:id
```
